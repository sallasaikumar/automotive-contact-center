## Emergency Backup

If something fails:

### Server Not Responding
1. Check terminal for errors
2. Restart: `npm start`
3. Wait 5 seconds
4. Refresh browser

### Features Not Loading
1. Check Network tab for errors
2. Verify API endpoints
3. Check server logs
4. Use manual feature clicks

### Auto-Demo Not Starting
1. Try Ctrl+D keyboard shortcut
2. Refresh page
3. Click features manually
4. Continue with manual demo
